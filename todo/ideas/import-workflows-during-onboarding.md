# Import Workflows During Onboarding

## What
Allow users to import existing CI/CD workflows into Everr as part of the onboarding flow.

## Why
New users may already have workflows in GitHub Actions or other systems. Importing them lowers the barrier to getting value from Everr immediately, instead of requiring manual re-creation from scratch.

## Who
New users going through onboarding who already have existing CI/CD workflows.
