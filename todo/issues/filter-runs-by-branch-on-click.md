# Filter runs by branch when a branch is clicked in the runs list

## What
Clicking a branch name in the runs list should filter the runs list to that branch.

## Where
Runs list UI — branch name elements.

## Steps to reproduce
1. Open the runs list
2. Click a branch name

## Expected
The runs list filters to show only runs for the clicked branch.

## Actual
Clicking a branch name does not filter the list.
