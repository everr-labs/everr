This guide assumes your organization already has access to a Everr workspace.

## First login checklist

1. Sign in to your Everr workspace.
2. Open the **Dashboard** at `/dashboard`.
3. Open the user menu and click **Connect GitHub**.
4. Follow the GitHub installation wizard and complete the app installation for the repositories you want to monitor.
5. Trigger or wait for a new pipeline run after installation.
6. Return to Everr, set a time window (for example, last 24h or last 7d), and confirm new run data appears in overview cards and recent runs.

Everr collects telemetry from new pipeline executions that happen after the GitHub App installation is completed.

## What you can do immediately

- Track overall CI success rate and trends.
- Drill into failing runs and failing jobs.
- Identify flaky tests and failure patterns.
- Analyze cost and runner utilization trends.
