**Route:** `/dashboard`

The main landing page provides a high-level summary of CI/CD pipeline health. It displays aggregate statistics and recent activity to give a quick pulse check across all repositories.

- Total job runs, successful runs, failed runs, and overall success rate stats
- Repository list with status and metrics
- Recent activity card showing daily pass/fail breakdown for the last 7 days
- Latest workflow runs
