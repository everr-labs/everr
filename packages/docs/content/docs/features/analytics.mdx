**Route:** `/dashboard/analytics`

Performance insights across all CI/CD pipelines. Four charts cover the key dimensions of build performance over a configurable time range.

- Duration trends chart (average, P50, P95 percentiles)
- Queue time analysis chart (wait time before jobs start)
- Success rate trends chart (build reliability over time)
- Runner utilization chart (most used runners and their metrics)
