---
title: Flaky Tests
description: Detection of tests with inconsistent pass/fail results.
---

**Route:** `/dashboard/flaky-tests`

Detects tests with inconsistent pass/fail results. The list view shows flaky test counts and trends, while the detail view breaks down a single test's failure rate by runner and execution history.

- Summary stats: flaky test count, total tests, flaky percentage
- Flakiness trend chart (flaky tests detected over time)
- Flaky tests table ranked by failure rate
- Per-test detail view (`/dashboard/flaky-tests/detail`) with runner breakdown and execution history timeline
