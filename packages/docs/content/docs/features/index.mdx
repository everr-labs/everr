Everr provides a comprehensive set of dashboard features for monitoring and analyzing your CI/CD pipelines. Each feature is designed to surface actionable insights from the OpenTelemetry data collected from your GitHub Actions workflows.

## Feature Overview

- [Dashboard Overview](/docs/features/dashboard) — High-level summary of CI/CD pipeline health
- [Runs List](/docs/features/runs) — Filterable, paginated table of all workflow runs
- [Run Detail](/docs/features/run-detail) — Detailed view of a single workflow run with job tree and trace waterfall
- [Cost Analysis](/docs/features/cost-analysis) — Estimated GitHub Actions spend based on runner usage
- [Test Results](/docs/features/test-results) — Aggregated test execution results across repositories
- [Repository Detail](/docs/features/repository-detail) — Repository-scoped metrics and activity

All pages include a time range picker that supports flexible [date math expressions](/docs/reference/datemath) for custom time filtering.
