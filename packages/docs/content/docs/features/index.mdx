Citric provides a comprehensive set of dashboard features for monitoring and analyzing your CI/CD pipelines. Each feature is designed to surface actionable insights from the OpenTelemetry data collected from your GitHub Actions workflows.

## Feature Overview

- [Dashboard Overview](/docs/features/dashboard) — High-level summary of CI/CD pipeline health
- [Runs List](/docs/features/runs) — Filterable, paginated table of all workflow runs
- [Run Detail](/docs/features/run-detail) — Detailed view of a single workflow run with job tree and trace waterfall
- [Analytics](/docs/features/analytics) — Performance insights across all CI/CD pipelines
- [Test Results](/docs/features/test-results) — Aggregated test execution results across repositories
- [Flaky Tests](/docs/features/flaky-tests) — Detection of tests with inconsistent pass/fail results
- [Failure Analysis](/docs/features/failure-analysis) — Clustering of recurring failure patterns across workflows
- [Repository Detail](/docs/features/repository-detail) — Repository-scoped metrics and activity
