title: Failure Analysis
description: Clustering of recurring failure patterns across workflows.
---

**Route:** `/dashboard/failures`

Clusters recurring failure patterns across workflows and tracks failure trends over time. Helps identify systemic issues by surfacing the most common error messages and most-affected repositories.

- Summary stats: total failures, unique patterns, most affected repo
- Failure trend chart (daily failures and unique patterns)
- Failure patterns table (recurring error messages ranked by frequency)
- Failures by repository table
