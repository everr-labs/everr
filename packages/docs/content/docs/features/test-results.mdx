**Route:** `/dashboard/test-results`

Aggregated test execution results across all repositories. Shows summary statistics, duration trends, and identifies the slowest tests for optimization.

- Summary stats: total tests, pass count, fail count, pass rate
- Test duration trend chart (average, P50, P95)
- Results by package table
- Slowest tests table (top 20 by average execution time)
