**Route:** `/dashboard/repos?name=owner/repo`

Repository-scoped metrics and activity. Loaded via query parameter, this page focuses on a single repository's build health, performance, and recent runs.

- Success rate chart (build reliability over time)
- Duration trends chart (P50 and P95 job duration)
- Top failing jobs table
- Active branches table
- Recent workflow runs for the repository
