**Route:** `/dashboard/runs`

A filterable, paginated table of all workflow runs. Supports narrowing results by multiple criteria to find specific runs quickly.

- Time range selector (7d, 14d, 30d, etc.)
- Multi-select filters for repository, branch, conclusion, workflow name, and run ID
- Paginated runs table (20 per page)
