## Authentication

- `everr login` ‚Äî authenticate with your Everr account
- `everr logout` ‚Äî sign out

## Setup

- `everr init` ‚Äî import runs and write assistant instructions for the current repository

## CI/CD Status

- `everr status` ‚Äî show the current commit's pipeline state (`state`, `active`, `completed`)
  - `--commit <sha>` target a specific commit
  - `--branch <name>` / `--repo <repo>` override repository context

- `everr watch` ‚Äî wait until the current `HEAD` commit finishes; exits non-zero if any run fails
  - `--commit <sha>` target a specific commit
  - `--attempt <n>` target a specific retry attempt
  - `--fail-fast` exit immediately when any run fails

- `everr runs` ‚Äî list recent runs
  - `--branch <name>` / `--current-branch` filter by branch
  - `--conclusion <success|failure|cancellation>` filter by outcome
  - `--workflow-name <name>` filter by workflow
  - `--run-id <id>` fetch a specific run

- `everr show <trace_id>` ‚Äî show jobs and steps for a run
  - `--failed` show only failed jobs and their failed steps

- `everr logs <trace_id>` ‚Äî print step logs (last 1000 lines by default)
  - `--job-name <job>` / `--step-number <n>` target a specific step
  - `--tail <n>` how many lines from the bottom
  - `--limit <n>` / `--offset <n>` raw paging

- `everr grep --pattern <text>` ‚Äî search failing step logs for a literal, case-insensitive pattern across the last 7 days, excluding the current branch
  - `--job-name <job>` / `--step-number <n>` target a specific step
  - `--branch <name>` scope to a branch

- `everr workflows` ‚Äî list workflows and their jobs
  - `--repo` / `--branch` filter results

Collection-style commands accept `--limit <n>` and `--offset <n>` for pagination.

## Local Telemetry

- `everr telemetry start` ‚Äî start the local OpenTelemetry collector
- `everr telemetry query "<SQL>"` ‚Äî query local telemetry with read-only SQL
- `everr telemetry endpoint` ‚Äî print the current local collector URL
- `everr wrap -- <command>` ‚Äî run a command normally and mirror stdout/stderr into local telemetry

## Test Analytics

- `everr test-history --test-name <name>` ‚Äî show historical executions for a specific test
  - `--module` disambiguate between tests with the same name

- `everr slowest-tests` ‚Äî list the slowest tests repo-wide
  - `--branch <name>` scope to a branch

## Other

- `everr ai-instructions` ‚Äî print the full CLI usage reference for AI assistants
- `everr uninstall` ‚Äî remove all config files, log out, and print the command to remove the CLI binary
