The Everr CLI is coming soon.

It is being designed for both AI agents and humans, so teams can move from CI/CD signals to action quickly without deep platform knowledge.

## Planned capabilities

- Get quick information about current CI status related to the latest pushed changes.
- Receive notifications when CI starts failing for the latest changes.
- Check when the last deploy was shipped.
- Get assisted guidance to fix failures and identify CI/CD performance bottlenecks.

## What this means in practice

You will be able to run focused CLI commands and use assistant integrations to:

- detect broken pipelines faster,
- reduce time spent digging through logs manually,
- and shorten time-to-fix for reliability and performance issues.

## Status

This feature is currently in progress and not available yet.
