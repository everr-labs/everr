The Everr CLI is coming soon.

It is being designed for both AI agents and humans, so teams can move from CI/CD signals to action quickly without deep platform knowledge.

## Value for teams

- Reduce time-to-awareness when recent changes impact CI/CD health.
- Keep everyone aligned on release status without switching between multiple tools.
- Shorten time-to-fix by turning failures and bottlenecks into guided actions.

## Value for AI agents and humans

- AI agents can reason over CI/CD context and suggest the next best actions.
- Humans can get clear, actionable status without deep CI/CD platform expertise.
- Both can collaborate in the same workflow, from detection to resolution.

## Install

Install the latest Everr CLI build from `main`:

```bash
curl -fsSL https://everr.dev/install.sh | sh
```

Then run the setup wizard:

```bash
everr install
```

- Installs latest `main` GitHub Release build (macOS arm64).
- CLI currently focuses on macOS for daemon + notifications.

## Status

This feature is currently in progress and available as a macOS arm64 preview.
