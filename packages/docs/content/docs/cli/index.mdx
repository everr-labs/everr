The Everr CLI helps AI agents and humans move quickly from CI/CD signal to action.

It is designed so teams can inspect failures, logs, and branch status from the current repository context without switching tools.

## Install

Install the latest Everr CLI build:

```bash
curl -fsSL https://everr.dev/install.sh | sh
```

The installer places `everr` in `~/.local/bin` and automatically runs the guided setup wizard to log in and configure your repository.

To configure a repo with Everr, run:

```bash
everr init
```

This command:
- Imports runs for the current repository
- Writes a managed block into `AGENTS.md` and/or `CLAUDE.md` (auto-detects which files exist; creates `AGENTS.md` if neither is present)

The snippet added to those files is:

```md
For CI, GitHub Actions, pipelines, workflow logs, or test performance tasks: call `everr ai-instructions` for full usage.

Quick start — run `everr status` to get the current commit's pipeline state while you plan your next steps.
```

This gives your AI assistant a minimal, always-present hint to reach for Everr when working on CI-related tasks.
