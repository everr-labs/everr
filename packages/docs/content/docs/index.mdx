title: Citric
description: CI/CD observability platform built on OpenTelemetry.
---

Citric is a CI/CD observability platform that collects telemetry from GitHub Actions workflows using OpenTelemetry and stores data in ClickHouse for visualization and analysis.

The platform consists of two main components:

- **Collector** (`collector/`) — Go-based OpenTelemetry Collector that receives CI webhook events and ingests workflow telemetry
- **Web Dashboard** (`packages/app`) — React + TanStack Start application for exploring pipeline data
