# Code Style

## Tooling

- **Formatter/Linter:** Biome (configured in `biome.json`)
- **Run:** `pnpm check` at root

## TypeScript/JavaScript

| Setting | Value |
|---------|-------|
| Indentation | Tabs |
| Quotes | Double quotes |
| Semicolons | Yes (Biome default) |

## Go

Standard Go conventions apply. Use `make lint-all` to check.
